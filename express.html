Express JS Hands-On Practice

Introduction to Express.js

Express.js is a popular web application framework for Node.js. It simplifies the process of building robust and scalable web applications. In this hands-on practice guide, we'll explore various aspects of Express.js.

Setting Up Your Development Environment

Before we dive into Express.js, make sure you have Node.js and npm installed on your system. You can download them from the official website: https://nodejs.org/.

Installing Express.js

                npm install express --save

Creating Your First Express Application

Let's start by creating a simple Express application. Create a new file named app.js and add the following code: 

                
const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('Hello, Express!');
});

app.listen(port, () => {
  console.log(`Server is running on port ${port}`);
});

Save the file and run the application using node app.js. You should see "Hello, Express!" when you visit http://localhost:3000 in your browser.

Handling Routes and Middleware

Express allows you to define routes and use middleware to handle requests. Here's an example of creating a route and using middleware for authentication: 

                
// Define a route
app.get('/about', (req, res) => {
  res.send('About Us');
});

// Middleware for authentication
function authenticate(req, res, next) {
  // Check authentication logic here
  if (authenticated) {
    next(); // Continue to the next middleware
  } else {
    res.status(401).send('Unauthorized');
  }
}

// Apply middleware to a route
app.get('/secure', authenticate, (req, res) => {
  res.send('Secure Page');
});

Working with Templates and Views

Express allows you to render dynamic views using template engines like EJS or Handlebars. Here's an example of using EJS for rendering views: 

                
// Set up EJS as the view engine
app.set('view engine', 'ejs');

// Create a route to render a dynamic page
app.get('/dynamic', (req, res) => {
  const data = {
    title: 'Dynamic Page',
    message: 'This is a dynamic page using EJS.'
  };
  res.render('dynamic', data);
});

Database Integration

Express can be integrated with various databases. One common choice is MongoDB. Here's an example of connecting Express to MongoDB using Mongoose: 

                
const mongoose = require('mongoose');
mongoose.connect('mongodb://localhost/mydatabase');

// Define a schema and model
const Schema = mongoose.Schema;
const ItemSchema = new Schema({
  name: String
});
const Item = mongoose.model('Item', ItemSchema);
